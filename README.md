# 免费节点每日多次更新
 
## 节点测速规则及上传更新规则

```
1.  测速环境为湖北移动 网络带宽100M(测速上限10Mb/s)
2.  测速线程为4
3.  v2ray和clash是相同节点 
```
# 说明
```
    节点全部来自互联网
    速度比我付费节点还快(不过极少数节点会失效)
```
